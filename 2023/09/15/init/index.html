初始化自己的个人博客

这里是我的第一篇文章，介绍如何使用github托管自己的个人博客网站;这里参考了两个不同的链接
First是本地创建自己的HEXO个人博客
Second是对第一个的补充操作，例如绑定GitHub,切换主题,使用自己的域名
Third是对第二个的补充操作，使用HEXO追加图片

Quick Start

Precondition

基础环境准备
1.需要注册自己的GitHub账号
2.电脑安装必须的组件
- node
- git

First Step

在github创建一个属于自己的仓库

  • 注意事项：
    • 创建时repository的名字要与github的名字相同，否则可能出现找不到css文件的情况：userName.github.io 通过setting中page将repository设置为github管理，然后等待部署完成就可以看到对应的页面啦

Second Step

使用git将项目拉取到本地

在本地创建一个目录：Github_blob
通过git命令将远程项目clone到本地

初始化博客项目

在git项目中新增一个目录(初始化HEXO时文件夹必须为空)：hexo
通过npm命令安装 hexo.cli并初始化Hexo项目

初始化后的目录层级结构

修改配置文件, _config.yml

本地启动

Tips：如果出现页面加载不出来，可能是端口被占用了。Ctrl+C 关闭服务器，运行 hexo server -p 5000 更改端口号后重试。

Third Step

上传到github

安装hexo-deployer-git

修改**_config.yml**配置文件

上传到GitHub(等待几分钟后部署完成就可以看到自己的效果了)
